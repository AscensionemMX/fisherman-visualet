# Fisherman Brand Core v1

## 1. Esencia de marca

Fisherman es una distribuidora farmacéutica moderna, confiable y cercana, diseñada para facilitar el abasto con control, tecnología y atención humana.

Fisherman no debe sentirse como una farmacia genérica ni como una empresa fría. Debe sentirse como una operación seria, fuerte y confiable que ayuda al cliente a comprar mejor.

## 2. Frase interna de marca

Tecnología, cercanía y confianza para una distribución farmacéutica más clara, rápida y controlada.

## 3. Personalidad

Fisherman debe sentirse:

- Seria.
- Confiable.
- Cercana.
- Tecnológica.
- Discreta.
- Fuerte.
- Ordenada.
- Precisa.
- Fácil de usar.
- Profesional.

## 4. Posicionamiento

Fisherman será percibida como una distribuidora farmacéutica con operación seria, atención cercana y herramientas digitales que facilitan la compra para clientes autorizados.

## 5. Promesa visual

Distribución farmacéutica con presencia, claridad y control.

## 6. Dirección estética

La identidad visual será elegante, dominante, farmacéutica y cinematográfica.

La website pública podrá sentirse oscura, premium y visualmente fuerte.

Visualet deberá sentirse más claro, simple, guiado y fácil de usar para clientes que no necesariamente dominan herramientas digitales.

## 7. Regla central de diseño

Oscuro para imponer.
Claro para vender.
Simple para que el cliente compre.

## 8. Paleta de marca

La paleta base será oscura, elegante y dominante, con acentos sanitarios y tecnológicos.

Colores principales:

- Negro profundo.
- Gris carbón.
- Gris claro.
- Blanco roto.
- Azul acero.
- Azul petróleo.
- Cian controlado.

## 9. Uso de color

### Website pública

La website pública usará fondos oscuros, acentos azul/cian, textos blancos y secciones cinematográficas.

Debe comunicar:

- Seriedad.
- Tecnología.
- Confianza.
- Dominio.
- Discreción.
- Profesionalismo.

### Visualet

Visualet usará fondos más claros, botones grandes, textos legibles y navegación simple.

Debe comunicar:

- Facilidad.
- Claridad.
- Rapidez.
- Compra guiada.
- Confianza.
- Cercanía.

## 10. Tipografía

La dirección tipográfica recomendada es:

- Titulares: Space Grotesk.
- Texto e interfaz: Inter.

Space Grotesk se usará para dar fuerza, modernidad y carácter.

Inter se usará para mantener legibilidad, especialmente en catálogo, botones, precios y formularios.

## 11. Logo

El logo actual de Fisherman se conserva por ahora.

Se desarrollarán versiones derivadas:

- Logo original completo.
- Logo para header.
- Logo claro para fondo oscuro.
- Logo oscuro para fondo claro.
- Isotipo del águila.
- Favicon.

## 12. Website pública

La website pública debe sentirse como la entrada cinematográfica al nuevo Fisherman.

Debe incluir:

- Hero fuerte.
- Logo visible.
- Mensaje claro.
- Sección de confianza.
- Sección de soluciones.
- Sección Visualet.
- Llamado para solicitar acceso.
- Footer discreto.

CTA significa Call To Action, o llamada a la acción.

## 13. Visualet

Visualet será el catálogo comercial inteligente de Fisherman.

Debe estar diseñado para clientes que necesitan comprar fácil, rápido y sin confundirse.

Debe incluir:

- Buscador grande.
- Categorías claras.
- Fotos de producto.
- Nombre del producto.
- Presentación.
- Laboratorio.
- Disponibilidad.
- Precio visible.
- Botón grande para agregar al carrito.
- Carrito simple.
- Enviar solicitud.
- Repetir pedido.
- Pedir ayuda.

## 14. Animaciones

La website pública podrá usar animaciones cinematográficas:

- Entrada suave del logo.
- Textos que aparecen por bloques.
- Tarjetas con movimiento en scroll.
- Líneas sutiles.
- Fondos con profundidad.
- Transiciones suaves entre secciones.

Visualet usará animaciones funcionales:

- Producto agregado al carrito.
- Carrito lateral.
- Botones con respuesta visual.
- Confirmaciones claras.
- Cargas suaves.
- Filtros fluidos.

## 15. Qué evitar

Fisherman no debe parecer:

- Farmacia genérica.
- Página infantil.
- Ecommerce barato.
- Marca saturada.
- Empresa informal.
- Diseño excesivamente médico común.
- Página oscura que dificulta comprar.
- Sistema complicado para clientes.
