# Frontend Stack — Fisherman Digital Core

## Objetivo

Definir la tecnología visual y de frontend para la website pública de Fisherman y Visualet.

## Website pública

La website pública se construirá con:

- Astro.
- TypeScript.
- React.
- Tailwind CSS.
- Motion.
- GSAP.

## Visualet

Visualet se construirá con:

- Astro.
- TypeScript.
- React.
- Tailwind CSS.
- Motion.
- Zustand.
- React Hook Form.
- Zod.

## Principio visual

Fisherman debe sentirse como una empresa seria, sanitaria, moderna, discreta y confiable.

Visualet debe sentirse como una herramienta comercial rápida, clara y útil para clientes y vendedores.

## Animaciones

La website pública podrá usar animaciones cinematográficas moderadas.

Visualet usará animaciones funcionales, enfocadas en claridad, velocidad y conversión.

## Relación con Dolibarr

Dolibarr es la herramienta principal.

Visualet será una capa comercial de catálogo, carrito y solicitudes.

En fase 1, Visualet no modificará Dolibarr.
